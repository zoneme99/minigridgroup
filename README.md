# Chess reinforcement learning
source: [Chess](https://python-chess.readthedocs.io/en/latest/)

### förslag på loss-function/reward system:
- Olika belöningar för olika pjäser man tar
- Behålla föregående drag som kan ha belönats för att sedan straffa pga en pjäsförlust
- belöningar för check
- belöningar för checkmate
- belöningar för att skydda/backa up pjäser
